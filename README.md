# sofia-utils

A utilities package named after my beautiful siamese cat

## Installation

### As an Editable Package

```bash
git clone https://github.com/luis-i-reyes-castro/sofia-utils.git
pip install -e sofia-utils/
```

### As a Dependency

Add this line to your `requirements.txt`:
```bash
sofia-utils @ git+https://github.com/luis-i-reyes-castro/sofia-utils.git@main
```

Re-run `pip`:
```bash
pip install -r requirements.txt
```
